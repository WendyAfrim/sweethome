# Sweet Home
Rent or buy houses online platform

## Getting Started
```bash
# clone git@github.com:WendyAfrim/sweethome.git
```

### Dependencies
* Git
* [composer](https://getcomposer.org/)
  

### Executing program
```bash
# cd /
# composer install
```
* In browser go to localhost:8081
